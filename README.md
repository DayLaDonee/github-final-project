# Simple Interest Calculator

## Overview

The **Simple Interest Calculator** is a lightweight and easy-to-use application designed to calculate the interest earned or paid on a principal amount over a fixed period of time using the simple interest formula. This project is ideal for students, beginners learning financial mathematics, or anyone who needs quick interest calculations.

Simple interest is commonly used in basic loans, savings accounts, and short-term financial planning.

---

## Purpose

The purpose of this calculator is to:

* Provide a fast and accurate way to calculate simple interest.
* Help users understand the relationship between principal, rate, and time.
* Serve as an educational project for learning programming fundamentals.
* Demonstrate the implementation of financial formulas in software.

---

## Formula Used

The calculator uses the standard simple interest formula:

[
SI = \frac{P \times R \times T}{100}
]

Where:

* **SI** = Simple Interest
* **P** = Principal amount (initial money)
* **R** = Annual interest rate (%)
* **T** = Time period (years)

The total amount after interest is:

[
A = P + SI
]

Where:

* **A** = Total amount

## Example

### Input:

Principal = 1000
Rate = 5%
Time = 2 years

### Calculation:

Simple Interest:

SI = (1000 × 5 × 2) / 100 = 100

Total Amount:

A = 1000 + 100 = 1100

### Output:

Simple Interest = 100
Total Amount = 1100
